# President Uhuru Kenyatta Cabinet Organogram (2013–2020)

Interactive visualization of President Uhuru Kenyatta’s cabinet structure across multiple government eras between 2013 and 2020.

## Live Demo

https://derrickmokaya.github.io/Organogram-President-Uhuru-Kenyatta-Cabinet/

---

## Overview

This project provides an interactive SVG-based organogram showing:

- Cabinet Secretaries
- Principal Secretaries
- Ministry structures
- Cabinet reshuffles across different years
- Search and filtering capabilities
- Hover tooltips with role details
- Multi-era timeline navigation

The visualization is fully browser-based and requires no backend setup.
